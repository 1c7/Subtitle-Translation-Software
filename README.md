## 补充说明

## 字幕文件的格式
* `.srt` 后缀的字幕文件是最简单的字幕格式，其中只有行数，时间(字幕的出现时间和消失时间), 以及字幕本身

* `.ass` 字幕可以定义字幕的样式，如大小，颜色，字体，位置。可定制能力更高
